How Does Phone Gap Work

The Nuts and Bolts

While Phone Gap is an incredible complex and andevanced piece of software (compilier), its slick desig allows for it to be easily used, even without a deep working understanding of the inner magic that creates the eventual phone app

CSS3, HTML5 and JavaScript underpin the whole effort

  • All of the coding will be done in these languages
  • Some nessecary steps are needed to prepare the raw code for the compiler

The Compiler is where the magic happens:

  • The raw HTML/CSS/JS is fed into the compilier
  • The code is compiled into a binary application archive
  • These files can then be downloaded onto a phone

Looks and feels like a native app

  • Aesthetically, Phone Gap allows apps to function like web browsers without a navigation tab at the top
  • Thereby, your web devolpment can be used to test your mobile devolpment

APIs and Plugins

There are numeroud APIs and plugins that serve to expidite and simplify the process of creating a mobile app.
These tools allow for the intergration of phone specific hardware (accelerometers), aswell as general hardware (microphones), and thervy allow for devolpers to create products that utilize the whole phone
